## 🐛 If Something's Wrong

### Can't see new design?
1. Force refresh: `Ctrl + Shift + R`
2. Try Incognito mode
3. Clear browser cache completely
4. Check you're on `/home.html` not `/index.html`

### Lessons don't load?
1. Open console: Press `F12`
2. Go to Console tab
3. Look for red errors
4. Screenshot and share with me

### Navigation broken?
1. Check URL in address bar
2. Should be `pillars-v3.html` not `pillars.html`
3. Old URL auto-redirects to new one
4. Force refresh if redirect fails
